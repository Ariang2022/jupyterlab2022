# Jupyter Lab

## Features

* Running in Ubuntu 18.04
* Aria2 WebUI/AriaNg included
* ffmpeg included
* handbrake included
* rclone
* mtn thumbnailer
* Modified from developeranaz/Jupyter-Notebook-Server-Heroku
